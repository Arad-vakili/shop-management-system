# 🛒 Shop Management System

[![Python Version](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

> A Python-based shop management system with order processing, discounts, tax calculation, and JSON storage.

## Features

- ✅ Order management (create, read, update, delete)
- ✅ Discount codes (SAVE10, OFF5, BLACK20)
- ✅ Tax calculation (IR: 9%, US: 7%, UK: 20%)
- ✅ Search by username, date, or price range
- ✅ Statistics and analytics
- ✅ JSON data persistence
- ✅ 100% test coverage
